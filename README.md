# **BlogHW14**

## Description
#### Post board with attempted commenting initialization. Users create a account and view what other users post and comments on. They can add and delete their own posts. Prompted to sign up or login on page load. Posts persists when user leaves the page and comes back to login in again.

## Installation
#### Download files from the git repo load into VS Code or similiar.  Open server.js in the integrated terminal. Run npm init and install required modules;"bcrypt": "^5.0.1","connect-session-sequelize": "^7.1.4","dotenv": "^16.0.1","express": "^4.18.1" "express-handlebars": "^6.0.6", "express-session": "^1.17.3", "handlebars": "^4.7.7", "mysql2": "^2.3.3", "nodemon": "^2.0.19",  "path": "^0.12.7", "router": "^1.3.7", "sequelize": "^6.21.3" then open intergeated MySQL in the terminal and run command SOURCE db/schema.sql; to get the database craeted. Ensure you are in the database with USE blog_db; then exit MySQL. Seed data base with the command npm run seed.  Open server  with nodemon or launch in heroku and open link then loging and create account.
